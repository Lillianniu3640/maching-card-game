# maching-card-game

## 遊戲介紹
1.使用HTML、CSS(3D翻牌效果)、JavaScript、jQuery<br>
2.針對兒童設計，可學習形狀、英文<br>
3.完成相同形狀配對即可結束<br>
4.遊戲設有音效、提示功能<br>
